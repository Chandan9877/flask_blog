About Flask Blog

Welcome to Flask Blog, a simple and modern blogging platform built using the Flask web framework. This project demonstrates how a full-stack web application can be developed with authentication, database integration, and responsive design.

✨ Features

  • 📝 Create, edit and delete blog posts
  • 🔐 Secure user authentication
  • 👤 User profile with profile pictures
  • 📧 Password reset via email
  • 📱 Responsive Bootstrap interface
  • 🗄️ SQLite database using SQLAlchemy

🛠 Technologies Used

  • Python
  • Flask
  • Flask-SQLAlchemy
  • Flask-WTF
  • Flask-Login
  • Flask-Mail
  • Jinja2 Templates
  • Bootstrap 4

🎯 Project Goal

The goal of this project is to explore Flask by building a practical blogging application that follows good development practices including modular design, authentication, database management, and clean user interfaces.

"Learning is best achieved by building real-world applications."
